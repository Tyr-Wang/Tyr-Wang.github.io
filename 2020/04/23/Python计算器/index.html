Python小破计算器

设计思路

观察win10自带的计算器，学习了它的部分处理方式

核心算法

从按键获取内容，存储到列表
计算（按等于键）拼接表达式
使用eval执行表达式，返回值回显

过滤非法表达

  • 过滤连续输入计算符号
    使用了两个标志位
    self.hasPressSign = False
    self.hasPressNum = False
    如果按下符号/= 时，标志hasPressSign+not hasPressNum：删除当前式子的最后一个符号/更新符号

括号实现

  1. 首先保证括号成对出现：使用了一个整型，左括号加入算式时+=1；右括号加入算式时-=1；==0时不能加右括号；!=0时不能执行等号（括号没有闭包）
  2. 合法情况，直接加入算式（append）
  3. 非法情况
    1. 当前输入屏幕无数字输入——忽略（啥也不做return）
    2. 当前输入屏幕有数字输入：
      1. 先加数字再加括号合法——加入算式（append）
      2. 加了数字也不合法——忽略（啥也不做return）

  • MR 把StringVar的值设置输入屏回去
  • MC 清空、M+: += 、M-: -=没有难点

防止destroy
