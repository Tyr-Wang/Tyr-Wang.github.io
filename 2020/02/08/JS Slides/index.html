JS Slides

之前看winesap大佬的课，看到大佬做的slides就十分心水，尤其是讲解Heap过程中，record&trace了内存空间的变化并简洁的复现到HTML上，清晰&炫酷——十分好懂

后来各种前端、神设计——跪感十足，遂学习JS

学以致用、以用促学，我发现了好多JS库可以简单实现我想要的功能，哇哦，I can really dance！

需求

最近录一个小课，可以试试制作效果炸裂的JS slides，更容易理解同时也方便在博客中符加展示

  • 可以顺滑的切换←↑↓→
  • 风格类似Markdown，最好直接支持Markdown语法（省事）

选择模块

  • impress.js
    • 突破传统边框束缚，全图切换，效果炫酷炸裂，甚至支持3D（比较心水的）；支持演示者模式
    • 不足：有点花里胡哨，使用较为复杂，风格适合 词云、极简风，没有内容慎用，容易喧宾夺主华而不实
    • Demo: http://impress.js.org
  • deck.js
    • 偏向传统横向ppt，不太markdown
    • 制作复杂，从底层写JS生成元素
    • 太久不维护，五年了
    • Demo: http://imakewebthings.com/deck.js
  • reveal.js
    • 跟想象中的最接近，极简风，全屏无框，效果简洁不花哨，支持上下左右滑动切换；额外附赠alt+click放大功能（贴心）；Markdown式用法，简单又兼容；还支持在线编辑；支持元素的特殊效果：渐入渐出，四方位平移，特性改变（颜色大小等HTML元素特性）；还有一个超好的特性，兼容插入Web内容到某一页作为背景，还是可以交互的，slide变成了一个悬浮前景frame；支持演示者模式
    • Demo: https://revealjs.com
  • remark.js
    • 极简部署，超兼容Markdown；支持演示者模式：注释、下一张幻灯片预览、时间
    • 不支持上下左右滑动切换，切换效果较简单；只支持整张幻灯片，不支持元素分别出现
    • Demo: remarkjs
  • Marp
    • 较为完备的Markdown Slides 平台，设计为易于在Github Pages上展示，支持 VS code 插件，制作方便；支持各种主题，支持KaTeX公式；
    • 较为庞大，是一个Marp生态；主题设置较为繁琐
    • Demo: Marp.app / Marp-cli-example
