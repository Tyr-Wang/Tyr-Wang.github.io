Some concepts

会话——Session

窗口——Window

tmux windows 不同于控制台窗口，更类似于tabs标签页

一个会话能被多个窗口接入，一个窗体可以开启多个窗口

why tmux

关闭终端（如，远程ssh网络连接突然中断）,再次打开时原终端里面的任务进程依然不会中断 ;

处于异地的两人可以对同一会话同步进行操作?

Usages

Sessions

tmux命令

tmux 默认开启一个新的Session并接入当前window

  • 新建会话 # tmux new [-s SESSION-NAME]

SESSION-NAME表示会话名称，可以为字母或数字

  • 查看会话 # tmux ls

  • 接入会话 # tmux a[ttach] -t [NUM or SESSION-NAME]

  • 分离会话 # tmux detach

  • 关闭会话 # tmux kill-session -t [SESSION-NAME]

在会话内部或外部执行均可

注 不能在tmux会话里面再新建会话，会报错 “sessions should be nested with care, unset $TMUX to force”
如果一定要创建，可以使用命令 ctrl+b :new [-s SESSION-NAME]，然后再ctrl+b s接入

  • 切换会话 # tmux switch -t [Num or SESSION-NAME]

  • 重命名会话 # tmux rename-session -t 0 [SESSION-NAME]

会话快捷键

Windows

窗口命令

  • 新建窗口 # tmux new-window -n [Win-Name]

  • 切换窗口 # tmux select-window -t [Win-Num or Win-Name]

  • 重命名窗口 # tmux rename-window [Win-Name]

窗口快捷键

Pane（split window)

窗格命令

以下命令都是在 Tmux 窗口中执行。

  • 划分窗格# tmux split-window [-h]

默认垂直分布（上下），-h 水平分布（左右）

  • 移动光标tmux select-pane [-U|-D|-L|-R]
  • 交换窗格位置 $ tmux swap-pane [-U|-D]
